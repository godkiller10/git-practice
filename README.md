**Objective:** This exercise covers a complete Git/GitHub workflow end-to-end — creating a GitHub repository, adding a C++ file (main.cpp) to it, cloning that repository locally, editing the file, and pushing the changes back to GitHub.

**Outcome:** By the end, the GitHub repository should contain an updated main.cpp with personal information (Name, Student ID, Section, plus one additional field of choice) — demonstrating a full clone → edit → add → commit → push cycle. The exercise also provides hands-on practice with six core Git commands: `git clone`, `git status`, `git diff`, `git add`, `git commit`, and `git push`.
